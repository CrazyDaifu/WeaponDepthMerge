# Security policy

This is an experimental graphics add-on loaded into a game's process. Only download binaries from a release or build them from the included source.

Report crashes, device-loss loops, corrupted rendering, unexpected file access, or suspicious binary differences through a private security report when the hosting platform supports it. Include the ReShade version, GPU, driver, game, API, and `ReShade.log`.

The project does not intentionally access the network, inject into unrelated processes, or read game data beyond graphics state exposed through the ReShade add-on API and Direct3D 9 interfaces.
