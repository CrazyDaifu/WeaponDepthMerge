# WeaponDepthMerge 1.0

Status: native D3D9 Battlefield 2 depth merge confirmed working.

Known issue: loading this add-on with Battlefield 2 + DXVK + Vulkan ReShade can cause a black screen and immediate process exit without a useful error. Version 1.1 addresses the cross-API startup crash.

The binary and `SHA256SUMS.txt` in this directory are frozen from the 1.0 baseline.
