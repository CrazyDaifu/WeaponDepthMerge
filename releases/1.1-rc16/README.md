# WeaponDepthMerge 1.1 RC16

Purpose: determine whether passive render-state tracking participates in WDM-002.

This build adds resource-view destruction, render-target/depth binding, viewport, vertex-buffer, and present callbacks to the RC15 lifecycle baseline. It has no draw, indexed-draw, depth-clear, effect-runtime, or overlay callbacks and performs no depth merging.

Status: diagnostic build; not a feature candidate.
